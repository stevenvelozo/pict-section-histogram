4. Sparse Numeric Series — Filling Gaps

Real-world data often has gaps. Here we demonstrate constructing bins that fill in missing years so the histogram shows continuous coverage. Use the buttons to toggle gap visibility. 

// Fill gaps in sparse year data before passing to histogram
function fillYearGaps(pSparseData) {
    var lookup = {};
    var minYear = Infinity, maxYear = -Infinity;
    pSparseData.forEach(function(b) {
        lookup[b.Label] = b.Value;
        minYear = Math.min(minYear, +b.Label);
        maxYear = Math.max(maxYear, +b.Label);
    });
    var filled = [];
    for (var y = minYear; y <= maxYear; y++) {
        filled.push({ Label: String(y), Value: lookup[y] || 0 });
    }
    return filled;
}

5. Dynamic Bar Colors Based on Value Ranges

Override BarColor per-render by dynamically assigning colors based on each bin's value. Uses onSelectionChange to recolor after each interaction. The technique: set each bin's color into a custom property and extend the renderer.

< 30   30–59   60–89   ≥ 90 

// Assign a BarColor property to each bin based on its value
function applyHeatColors(pBins) {
    pBins.forEach(function(b) {
        if (b.Value < 30)       b.BarColor = '#E74C3C';
        else if (b.Value < 60)  b.BarColor = '#F39C12';
        else if (b.Value < 90)  b.BarColor = '#27AE60';
        else                     b.BarColor = '#2980B9';
    });
}

8. Programmatic API

Use methods like setBins(), setSelection(), setOrientation(), and getSelection() to control the histogram from code.

Use the buttons above to call API methods.
// Programmatic control
histView.setBins([{ Label: 'A', Value: 10 }, ...]);
histView.setSelection({ Start: 1, End: 3 });
histView.setOrientation('horizontal');
var sel = histView.getSelection();
// sel = { Mode:'range', RangeStart:1, RangeEnd:3, SelectedIndices:[1,2,3], ... }
