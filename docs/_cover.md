# Pict Section Histogram

> Interactive histogram visualizations for Pict applications

Render vertical or horizontal bar charts from an array of label/value bins. Supports three rendering targets (browser HTML, terminal UI, CLI) and three selection modes (single click, multi-toggle, range slider). Configure layout, colors, and behavior entirely in JSON.

- **Three Render Modes** -- Browser HTML/CSS, ConsoleUI text art, CLI ANSI output
- **Vertical & Horizontal** -- Switch orientation at runtime
- **Range Selection** -- Draggable slider handles for contiguous range filtering
- **Data Binding** -- Read bins from AppData, write selection state back automatically
- **Config-Driven** -- All options controlled via JSON configuration
- **Extensible** -- Override selection callbacks, swap renderers, subclass for custom behavior
